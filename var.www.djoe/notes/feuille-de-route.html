Remarques & idées

Combien d'utilisateurs ? Une estimation est nécéssaire
Readme Github de CoTURN: plusieurs miliers d'appels vocaux par processeur en TURN, +ieurs dizaines de milliers en STUN
Pour optimiser la prise en charge TURN le trafic TLS doit passer par 443 donc problème si le serveur TURN est installé sur la même machine que le serveur HTTPS

Menu d'interface

Sera constituée de deux types d'objets:
  • Des groupes d'éléments d'interface, représentables en menus
  • Des élements, qui contiendront:
      • Une étiquette
      • Une description courte
      • Une description longue
      • Un sub-division d'interface

Devra contenir un indicateur de position en tête (ex: Paramètres / ...)
Tous les élements seront listés pour permettre une recherche.

Structure possible pour le menu principal:
  • Rechercher: paramètres, utilisateurs, messages, ...
  • Disponibilité
      • Occupé, en ligne, ....
  • Notifications
      • Muet
      • Activer les notifications de bureau
  • Discussions
      • Créer une discussion de groupe
      • Liste des discussions de groupe
      • Rejoindre une discussion de groupe
  • Etherpad
      • Création
      • Liste
