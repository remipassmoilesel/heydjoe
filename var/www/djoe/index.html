Hey Djoe !

Messagerie instantanée multimédia destinée au portail collaboratif Silverpeas
La messagerie est aussi disponible intégrée au portail Silverpeas 6

Bienvenue !

Cette espace est dédié à l'essai et au développement de Djoe, un système complet de messagerie instantanée multimédia destiné à l'intégration. Ce système est disponible sous licence GPLv3: vous pouvez copier, modifier et distribuer ce matériel librement !

Attention:

  • Préférez l'utillisation d'un compte aléatoire: si vous utilisez un des premiers comptes prédéfinis disponibles, il est possible que quelqu'un d'autre l'utilise simultanément
  • Les outils utilisés sont modernes, votre navigateur doit absolument être à jour pour utiliser ce système
  • Le travail mis à disposition est en constante évolution, par conséquent les serveurs peuvent être momentanément indisponibles
  • Cette plateforme est en accès public: ne l'utilisez pas pour des communications confidentielles

Fonctionnalités

  • Messagerie instantanée textuelle un pour un, un pour plusieurs
  • Edition collaborative de documents texte en temps réel
  • Appels vidéo et vidéoconférence jusqu'à six participants
  • Partage d'écran un pour un, un pour plusieurs
Ce système à été développé grâce à de formidables projets libres:
  • JSXC: un client web XMPP modifié permettant la messagerie instantanée texte et multimédia
  • Etherpad: une solution d'édition en ligne collaborative
  • Openfire: un serveur XMPP libre
  • rfc5766-turn-server: un serveur relai multimédia

Intégration

Coté client

Pour intégrer le client, vous devez tout d'abord installer les logiciels serveur nécessaires. Puis sur la page web destination: 

<!-- XMPP client dependencies -->

<link href="bower_components/jquery-ui/themes/base/jquery-ui.css" media="all" rel="stylesheet" type="text/css"/>
<script src="bower_components/jquery/dist/jquery.js"></script>
<script src="bower_components/jquery-ui/jquery-ui.js"></script>

<!-- end XMPP client dependencies -->

<!-- XMPP client -->

<link href="jsxc/build/css/jsxc.css" media="all" rel="stylesheet" type="text/css"/>
<script src="jsxc/build/lib/jsxc.dep.js"></script>
<script src="jsxc/build/jsxc.js"></script>
<script src="jsxc/build/jsxc_init.js"></script>

<!-- end XMPP client -->

Coté serveur

Ce système nécessite plusieurs logiciels serveur. Une installation test est disponible sous la forme d'une image Docker: 

  # /!\ Nécéssite Docker: https://docs.docker.com/engine/installation/

  $ git clone https://github.com/remipassmoilesel/djoe-docker && cd djoe-docker && ./build-and-launch.sh
