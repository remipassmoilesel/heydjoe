Presentation

Hey Djoe ! is a free multimedia instant messaging system. A small presentation video is available here: https://www.youtube.com/embed/dMTQm9NEYTs

Install your own server in 3 minutes

Example on Ubuntu 16.04: 

  # /!\ Need Docker, see https://docs.docker.com/engine/installation/

  $ git clone https://github.com/remipassmoilesel/heydjoe-docker
  $ cd djoe-docker
  $ ./build-and-launch.sh

Integration

Once the server is installed , you can integrate the client into your web page with a few lines of code : 

<!-- XMPP client dependencies -->

<link href="bower_components/jquery-ui/themes/base/jquery-ui.css" media="all" rel="stylesheet" type="text/css"/>
<script src="bower_components/jquery/dist/jquery.js"></script>
<script src="bower_components/jquery-ui/jquery-ui.js"></script>

<!-- end XMPP client dependencies -->

<!-- XMPP client -->

<link href="jsxc/build/css/jsxc.css" media="all" rel="stylesheet" type="text/css"/>
<script src="jsxc/build/lib/jsxc.dep.js"></script>
<script src="jsxc/build/jsxc.js"></script>
<script src="jsxc/build/jsxc_init.js"></script>

<!-- end XMPP client -->
