Installer l'extension Chrome de capture d'écran

Pour pouvoir capturer et partager votre écran suivez attentivement les étapes suivantes. Ces opérations ne sont pas nécéssaires pour recevoir le flux d'un autre écran.

  1. Ouvrez cette boite de dialogue dans le navigateur Chrome
  2. Cliquez ici pour télécharger l'extension puis enregistrez là dans le dossier "Téléchargement" de votre système
  3. Cliquez ici pour ouvrir un nouvel onglet
  4. Puis saisissez dans la barre d'adresse: chrome://extensions
  5. Puis glissez-déposez le fichier "#######.crx" que vous avez téléchargé sur la page de paramétrage
  6. Acceptez l'installation de l'extension. Une fois l'extension installée, vous pouvez supprimer le fichier téléchargé.
  7. Enfin rechargez la page en cliquant ci-dessous
